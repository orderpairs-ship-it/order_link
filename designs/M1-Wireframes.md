## 产品概述

**产品名称:** 订单转换助手 (Order Conversion Assistant)

**核心价值:** 将外部客户系统的订单数据自动抓取并转换为内部 ERP 格式

**用户角色:** 运营人员/订单管理员

**核心流程:**
```
登录客户系统 → 抓取订单 → 字段映射 → 物料映射 → 提交到 ERP
```

### 组件说明

| 区域 | 名称 | 功能 |
|------|------|------|
| 顶部栏 | Header | 产品名称 + 用户信息 + 全局设置入口 |
| 左侧 | 快捷操作区 | 高频功能快速入口 |
| 右上 | 今日概览 | 关键指标卡片（待处理/进行中/已完成） |
| 右下 | 订单列表 | 订单表格 + 分页 + 状态筛选 |

### 状态定义

| 状态 | 图标 | 说明 |
|------|------|------|
| 待映射 | 🟡 黄色 | 订单已抓取，等待字段/物料映射 |
| 进行中 | 🔵 蓝色 | 正在转换/提交中 |
| 已完成 | 🟢 绿色 | 成功提交到 ERP |
| 失败 | 🔴 红色 | 转换或提交失败，需人工介入 |

### 交互说明

1. **点击订单行** → 展开订单详情面板（右侧滑出）
2. **点击 ▶ 操作按钮** → 进入转换流程
3. **点击 [+ 新任务]** → 创建新抓取任务
4. **点击 [批量转换]** → 多选订单批量处理

### 交互说明

1. **外部字段列表** → 从客户系统自动拉取，只读
2. **内部字段下拉** → 选择 ERP 对应字段，支持搜索
3. **必填字段标识** → 未配置的必填字段显示红色警告
4. **[自动匹配]** → 基于字段名相似度智能推荐映射
5. **[保存]** → 验证必填字段 → 保存映射关系

### 映射规则

| 规则类型 | 说明 |
|----------|------|
| 直接映射 | 1:1 字段对应（如 order_no → ERP_Order_ID） |
| 拆分映射 | 1 个外部字段 → 多个内部字段（如地址拆分） |
| 合并映射 | 多个外部字段 → 1 个内部字段（如姓 + 名） |
| 转换映射 | 需要格式转换（如日期格式、币种换算） |

### 物料匹配策略

| 优先级 | 策略 | 说明 |
|--------|------|------|
| P0 | 精确匹配 | 客户编码 = ERP 编码 |
| P1 | 名称匹配 | 物料名称相似度 > 90% |
| P2 | 规格匹配 | 规格参数一致（如 M3x10） |
| P3 | 人工匹配 | 运营手动选择 |

## 交互流程

### 完整转换流程

```
┌─────────────┐
│  主界面     │
│  选择订单   │
└──────┬──────┘
       │ 点击 [转换]
       ▼
┌─────────────┐     是      ┌─────────────┐
│  检查映射   │────────────→│  执行转换   │
│  是否配置   │             │             │
└──────┬──────┘             └──────┬──────┘
       │ 否                        │
       ▼                           ▼
┌─────────────┐             ┌─────────────┐
│  字段映射   │             │  提交 ERP   │
│  弹窗       │             │             │
└──────┬──────┘             └──────┬──────┘
       │                           │
       ▼                           ▼
┌─────────────┐             ┌─────────────┐
│  物料映射   │             │  显示结果   │
│  弹窗       │             │  (成功/失败)│
└──────┬──────┘             └─────────────┘
       │
       ▼
   [保存映射]
       │
       └─────────────────────────→ 返回转换流程
```

### 状态流转

```
待映射 ──→ 进行中 ──→ 已完成
  │          │
  │          └────→ 失败 (可重试)
  │
  └────→ 配置映射后自动开始
```
