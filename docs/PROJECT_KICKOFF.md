## 已确认需求

### 用户场景
销售人员每天需要：
1. 登录多个客户订单系统（BYD/华为/美团等）
2. 查询与自己公司相关的订单
3. 将外部订单转换为内部订单并提交

### 技术约束
| 项目 | 说明 |
|------|------|
| 外部系统对接 | 网页爬虫（无 API） |
| 内部系统对接 | 网页形式录入 |
| 认证方式 | 用户名 + 密码 + 手机验证码 |
| 字段映射 | 无现成映射，需可配置 |
| 用户规模 | 单机使用，4-5 单/天 |

### UI 方向
- 左右分栏工作台式设计
- 左侧：外部订单源数据
- 右侧：内部订单草稿
- 支持智能匹配 + 人工干预
